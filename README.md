# Freeloop's Blog

技术博客，分享编程与思考。

## 访问

- 博客主页：https://freeloop001.github.io/blog

## 编写文章

在 `_posts/` 目录下创建 Markdown 文件，文件名格式：

```
YYYY-MM-DD-title.md
```

文章头部添加元信息：

```yaml
---
layout: post
title: "文章标题"
date: 2026-03-11
tags: [tag1, tag2]
---
```

## 本地预览

```bash
bundle install
bundle exec jekyll serve
```

然后访问 http://127.0.0.1:4000
