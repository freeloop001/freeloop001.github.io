## 1. 本地模型 (Ollama)

### 安装和使用

```bash
# 安装 Ollama
# macOS: brew install ollama
# Linux: curl -fsSL https://ollama.com/install.sh | sh

# 拉取模型
ollama pull llama2
ollama pull codellama
ollama pull mistral
ollama pull qwen:7b

# 列出模型
ollama list

# 运行模型
ollama run llama2
```

### Python API

```python
import ollama

# 对话
response = ollama.chat(
    model='llama2',
    messages=[
        {'role': 'user', 'content': '你好'}
    ]
)

print(response['message']['content'])

# 流式输出
for chunk in ollama.chat(
    model='llama2',
    messages=[{'role': 'user', 'content': '写一首诗'}],
    stream=True
):
    print(chunk['message']['content'], end='')

# 生成 Embedding
response = ollama.embeddings(
    model='nomic-embed-text',
    prompt='Python是一门编程语言'
)
print(response['embedding'][:10])
```

### Ollama vs OpenAI

| 特性 | Ollama | OpenAI API |
|------|--------|------------|
| 部署方式 | 本地 | 云端 |
| 隐私 | 完全隐私 | 数据上传 |
| 成本 | 硬件成本 | 按调用付费 |
| 速度 | 取决于硬件 | 速度快 |

## 2. 生产级模型 (vLLM)

### 快速开始

```bash
# 安装 vLLM
pip install vllm

# 启动服务
vllm serve Qwen/Qwen2-7B-Instruct --dtype half

# 或使用 OpenAI 兼容模式
vllm serve Qwen/Qwen2-7B-Instruct --api-key openai
```

### Python API

```python
from openai import OpenAI

# 连接 vLLM
client = OpenAI(
    base_url="http://localhost:8000/v1",
    api_key="EMPTY"
)

# 对话
response = client.chat.completions.create(
    model="qwen",
    messages=[{"role": "user", "content": "你好"}]
)

print(response.choices[0].message.content)
```

### vLLM 特点

- **PagedAttention**：高效显存管理
- **Continuous Batching**：连续批处理
- **Tensor Parallelism**：张量并行
- **OpenAI 兼容**：API 兼容

## 3. 模型量化与优化

### 量化类型

| 类型 | 精度 | 显存减少 | 速度提升 | 效果损失 |
|------|------|----------|----------|----------|
| FP16 | 16位 | ~50% | 1x | 最小 |
| INT8 | 8位 | ~75% | ~2x | 较小 |
| INT4 | 4位 | ~90% | ~4x | 中等 |
| GPTQ | 4位 | ~90% | ~4x | 较小 |
| AWQ | 4位 | ~90% | ~4x | 最小 |

### 使用 GGUF 量化

```bash
# 使用 llama.cpp 量化
# 安装
pip install llama-cpp-python

# 量化命令
./quantize /path/to/model.gguf /path/to/model-q4_k_m.gguf q4_k_m
```

### 使用 GPTQ 量化

```bash
# 安装
pip install auto-gptq optimum

# 量化
from transformers import AutoModelForCausalLM, AutoTokenizer, GPTQQuantizer

model = AutoModelForCausalLM.from_pretrained(
    "model/name",
    device_map="auto"
)

quantizer = GPTQQuantizer(quantization_config={"bits": 4})
quantized_model = quantizer.quantize_model(model, tokenizer)

quantized_model.save_pretrained("model-name-4bit")
```

### vLLM 量化配置

```bash
# 启动 vLLM 使用量化模型
vllm serve meta-llama/Llama-2-7b-chat-hf \
    --dtype half \
    --quantization awq \
    --tensor-parallel-size 2
```

### LoRA 微调优化

```python
from peft import LoraConfig, get_peft_model
from transformers import AutoModelForCausalLM

# 加载基础模型
model = AutoModelForCausalLM.from_pretrained(
    "meta-llama/Llama-2-7b",
    load_in_8bit=True
)

# 配置 LoRA
lora_config = LoraConfig(
    r=16,  # LoRA rank
    lora_alpha=32,
    target_modules=["q_proj", "v_proj"],
    lora_dropout=0.05,
    bias="none",
    task_type="CAUSAL_LM"
)

# 应用 LoRA
model = get_peft_model(model, lora_config)
model.print_trainable_parameters()
# 输出: trainable params: 4,194,304 || all params: 6,742,609,920 || trainable%: 0.062
```

## 4. 分布式部署

### Docker 部署

```dockerfile
# Dockerfile
FROM nvidia/cuda:12.1-runtime-ubuntu22.04

WORKDIR /app

# 安装 vLLM
RUN pip install vllm

# 复制模型（可选：使用 volume 挂载）
COPY ./models /models

# 启动命令
CMD ["vllm", "serve", "/models/llama2", "--dtype", "half"]
```

```yaml
# docker-compose.yml
version: '3.8'

services:
  vllm:
    image: vllm/vllm-openai:latest
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: 1
              capabilities: [gpu]
    ports:
      - "8000:8000"
    volumes:
      - ./models:/models
    environment:
      - NVIDIA_VISIBLE_DEVICES=all
    command: vllm serve /models/llama2 --dtype half
```

### 负载均衡

```python
from openai import OpenAI
import random

class LoadBalancer:
    def __init__(self, endpoints: list):
        self.endpoints = endpoints

    def create_client(self):
        endpoint = random.choice(self.endpoints)
        return OpenAI(
            base_url=f"http://{endpoint}/v1",
            api_key="dummy"
        )

    def call(self, prompt: str):
        client = self.create_client()
        response = client.chat.completions.create(
            model="llama2",
            messages=[{"role": "user", "content": prompt}]
        )
        return response

# 使用
lb = LoadBalancer(["192.168.1.10:8000", "192.168.1.11:8000", "192.168.1.12:8000"])
result = lb.call("你好")
```

## 6. 可观测性 (LangSmith)

### 快速开始

```bash
# 安装
pip install langsmith

# 设置环境变量
export LANGCHAIN_TRACING_V2=true
export LANGCHAIN_API_KEY="your-key"
```

### Python 使用

```python
from langchain_openai import ChatOpenAI
from langchain.prompts import PromptTemplate
from langchain.chains import LLMChain
from langsmith import traceable

# 追踪函数
@traceable
def my_function(text):
    # 你的代码
    return text.upper()

# 追踪 Chain
llm = ChatOpenAI(api_key="your-key")
prompt = PromptTemplate.from_template("{product}的创始人是谁？")
chain = LLMChain(llm=llm, prompt=prompt)

# 执行（自动追踪）
result = chain.invoke({"product": "Apple"})
print(result)
```

## 7. 工作流编排 (n8n)

### n8n 简介

n8n 是一个强大的工作流自动化工具，支持：
- 视觉化流程编排
- 300+ 集成
- 自定义节点
- 本地部署

### 基本概念

- **Trigger**：触发器
- **Node**：节点
- **Connection**：连接
- **Workflow**：工作流

### n8n + AI 示例

```json
// n8n 工作流 JSON
{
  "nodes": [
    {
      "name": "Chat Trigger",
      "type": "n8n-nodes-base.chatTrigger",
      "parameters": {}
    },
    {
      "name": "OpenAI",
      "type": "openai",
      "parameters": {
        "operation": "chat",
        "model": "gpt-4",
        "prompt": "={{ $json.message }}"
      }
    },
    {
      "name": "Respond to Chat",
      "type": "n8n-nodes-base.respondToChat",
      "parameters": {
        "text": "={{ $json.choices[0].message.content }}"
      }
    }
  ],
  "connections": {
    "Chat Trigger": {
      "main": [[{"node": "OpenAI"}]]
    },
    "OpenAI": {
      "main": [[{"node": "Respond to Chat"}]]
    }
  }
}
```

## 8. 实战项目

### 项目：本地 LLM 服务

```python
# 使用 Ollama 部署本地模型服务
# server.py

import ollama
from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route('/chat', methods=['POST'])
def chat():
    data = request.json
    response = ollama.chat(
        model=data.get('model', 'llama2'),
        messages=[{'role': 'user', 'content': data['message']}]
    )
    return jsonify({
        'response': response['message']['content']
    })

@app.route('/models', methods=['GET'])
def models():
    models = ollama.list()
    return jsonify({'models': models['models']})

if __name__ == '__main__':
    app.run(port=5000)
```

### 项目：模型路由系统

```python
# 使用 LiteLLM 实现智能路由
from litellm import completion
import os

class ModelRouter:
    def __init__(self):
        self.routing_rules = {
            "simple": "gpt-3.5-turbo",
            "complex": "gpt-4",
            "code": "anthropic/claude-3-sonnet-20240229",
            "local": "ollama/codellama"
        }

    def route(self, task_type: str, messages: list):
        model = self.routing_rules.get(task_type, "gpt-3.5-turbo")
        return completion(model=model, messages=messages)

# 使用
router = ModelRouter()

# 简单问题
result = router.route("simple", [{"role": "user", "content": "你好"}])

# 复杂问题
result = router.route("complex", [{"role": "user", "content": "解释量子计算"}])

# 写代码
result = router.route("code", [{"role": "user", "content": "写一个排序算法"}])
```
