## 1. 本地模型 (Ollama)

### 安装和使用

```bash
# 安装 Ollama
# macOS: brew install ollama
# Linux: curl -fsSL https://ollama.com/install.sh | sh

# 拉取模型
ollama pull llama2
ollama pull codellama
ollama pull mistral
ollama pull qwen:7b

# 列出模型
ollama list

# 运行模型
ollama run llama2
```

### Python API

```python
import ollama

# 对话
response = ollama.chat(
    model='llama2',
    messages=[
        {'role': 'user', 'content': '你好'}
    ]
)

print(response['message']['content'])

# 流式输出
for chunk in ollama.chat(
    model='llama2',
    messages=[{'role': 'user', 'content': '写一首诗'}],
    stream=True
):
    print(chunk['message']['content'], end='')

# 生成 Embedding
response = ollama.embeddings(
    model='nomic-embed-text',
    prompt='Python是一门编程语言'
)
print(response['embedding'][:10])
```

### Ollama vs OpenAI

| 特性 | Ollama | OpenAI API |
|------|--------|------------|
| 部署方式 | 本地 | 云端 |
| 隐私 | 完全隐私 | 数据上传 |
| 成本 | 硬件成本 | 按调用付费 |
| 速度 | 取决于硬件 | 速度快 |

## 2. 生产级模型 (vLLM)

### 快速开始

```bash
# 安装 vLLM
pip install vllm

# 启动服务
vllm serve Qwen/Qwen2-7B-Instruct --dtype half

# 或使用 OpenAI 兼容模式
vllm serve Qwen/Qwen2-7B-Instruct --api-key openai
```

### Python API

```python
from openai import OpenAI

# 连接 vLLM
client = OpenAI(
    base_url="http://localhost:8000/v1",
    api_key="EMPTY"
)

# 对话
response = client.chat.completions.create(
    model="qwen",
    messages=[{"role": "user", "content": "你好"}]
)

print(response.choices[0].message.content)
```

### vLLM 特点

- **PagedAttention**：高效显存管理
- **Continuous Batching**：连续批处理
- **Tensor Parallelism**：张量并行
- **OpenAI 兼容**：API 兼容

## 3. 模型路由 (LiteLLM)

### 统一 API

```python
from litellm import completion

# 切换模型无需修改代码
response = completion(
    model="gpt-4",
    messages=[{"role": "user", "content": "你好"}]
)

# 切换到 Claude
response = completion(
    model="claude-3-sonnet-20240229",
    messages=[{"role": "user", "content": "你好"}]
)

# 切换到本地模型
response = completion(
    model="ollama/llama2",
    messages=[{"role": "user", "content": "你好"}]
)
```

### 配置代理

```yaml
# litellm_config.yaml
model_list:
  - model_name: gpt-4
    litellm_params:
      model: openai/gpt-4
      api_key: os.environ/OPENAI_API_KEY

  - model_name: claude
    litellm_params:
      model: anthropic/claude-3-sonnet-20240229
      api_key: os.environ/ANTHROPIC_API_KEY
```

## 4. 可观测性 (LangSmith)

### 快速开始

```bash
# 安装
pip install langsmith

# 设置环境变量
export LANGCHAIN_TRACING_V2=true
export LANGCHAIN_API_KEY="your-key"
```

### Python 使用

```python
from langchain_openai import ChatOpenAI
from langchain.prompts import PromptTemplate
from langchain.chains import LLMChain
from langsmith import traceable

# 追踪函数
@traceable
def my_function(text):
    # 你的代码
    return text.upper()

# 追踪 Chain
llm = ChatOpenAI(api_key="your-key")
prompt = PromptTemplate.from_template("{product}的创始人是谁？")
chain = LLMChain(llm=llm, prompt=prompt)

# 执行（自动追踪）
result = chain.invoke({"product": "Apple"})
print(result)
```

## 5. 工作流编排 (n8n)

### n8n 简介

n8n 是一个强大的工作流自动化工具，支持：
- 视觉化流程编排
- 300+ 集成
- 自定义节点
- 本地部署

### 基本概念

- **Trigger**：触发器
- **Node**：节点
- **Connection**：连接
- **Workflow**：工作流

### n8n + AI 示例

```json
// n8n 工作流 JSON
{
  "nodes": [
    {
      "name": "Chat Trigger",
      "type": "n8n-nodes-base.chatTrigger",
      "parameters": {}
    },
    {
      "name": "OpenAI",
      "type": "openai",
      "parameters": {
        "operation": "chat",
        "model": "gpt-4",
        "prompt": "={{ $json.message }}"
      }
    },
    {
      "name": "Respond to Chat",
      "type": "n8n-nodes-base.respondToChat",
      "parameters": {
        "text": "={{ $json.choices[0].message.content }}"
      }
    }
  ],
  "connections": {
    "Chat Trigger": {
      "main": [[{"node": "OpenAI"}]]
    },
    "OpenAI": {
      "main": [[{"node": "Respond to Chat"}]]
    }
  }
}
```

### Dify

### Flowise

## 6. 实战项目

### 项目：本地 LLM 服务

```python
# 使用 Ollama 部署本地模型服务
# server.py

import ollama
from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route('/chat', methods=['POST'])
def chat():
    data = request.json
    response = ollama.chat(
        model=data.get('model', 'llama2'),
        messages=[{'role': 'user', 'content': data['message']}]
    )
    return jsonify({
        'response': response['message']['content']
    })

@app.route('/models', methods=['GET'])
def models():
    models = ollama.list()
    return jsonify({'models': models['models']})

if __name__ == '__main__':
    app.run(port=5000)
```

### 项目：模型路由系统

```python
# 使用 LiteLLM 实现智能路由
from litellm import completion
import os

class ModelRouter:
    def __init__(self):
        self.routing_rules = {
            "simple": "gpt-3.5-turbo",
            "complex": "gpt-4",
            "code": "anthropic/claude-3-sonnet-20240229",
            "local": "ollama/codellama"
        }

    def route(self, task_type: str, messages: list):
        model = self.routing_rules.get(task_type, "gpt-3.5-turbo")
        return completion(model=model, messages=messages)

# 使用
router = ModelRouter()

# 简单问题
result = router.route("simple", [{"role": "user", "content": "你好"}])

# 复杂问题
result = router.route("complex", [{"role": "user", "content": "解释量子计算"}])

# 写代码
result = router.route("code", [{"role": "user", "content": "写一个排序算法"}])
```
