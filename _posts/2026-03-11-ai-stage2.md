## 1. OpenAI API

### 快速开始

```python
import os
from openai import OpenAI

# 方式1: 环境变量
# export OPENAI_API_KEY="your-api-key"
client = OpenAI()

# 方式2: 直接传入
client = OpenAI(api_key="your-api-key")

# 基础对话
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "你是一个有帮助的助手"},
        {"role": "user", "content": "用一句话介绍Python"}
    ]
)

print(response.choices[0].message.content)
```

### 流式输出

```python
from openai import OpenAI

client = OpenAI()

stream = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "写一首关于春天的诗"}],
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="")
```

### 支持的模型

| 模型 | 特点 | 适用场景 |
|------|------|----------|
| gpt-4o | 最新旗舰多模态 | 复杂任务 |
| gpt-4-turbo | 高性价比 | 日常开发 |
| gpt-3.5-turbo | 快速、便宜 | 简单对话 |

## 2. Anthropic API (Claude)

### 快速开始

```python
from anthropic import Anthropic

client = Anthropic(api_key="your-api-key")

message = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "用一句话介绍Python"}
    ]
)

print(message.content[0].text)
```

### 流式输出

```python
from anthropic import Anthropic

client = Anthropic()

with client.messages.stream(
    model="claude-sonnet-4-20250514",
    max_tokens=1024,
    messages=[{"role": "user", "content": "解释什么是RAG"}]
) as stream:
    for text in stream.text_stream:
        print(text, end="")
```

### Claude 模型选择

| 模型 | 能力 | 速度 | 推荐场景 |
|------|------|------|----------|
| claude-opus-4-6 | 最强 | 慢 | 复杂推理 |
| claude-sonnet-4-6 | 均衡 | 中 | 日常开发 |
| claude-haiku-4-5 | 基础 | 快 | 简单任务 |

## 4. Prompt 设计

### System Prompt

```python
# 定义 AI 角色和能力
system_prompt = """你是一个专业的Python编程助手。
- 擅长写出简洁、高效的Python代码
- 代码遵循 PEP 8 规范
- 如果用户请求写代码，先解释思路再给出代码"""

messages = [
    {"role": "system", "content": system_prompt},
    {"role": "user", "content": "如何快速排序？"}
]
```

### Few-shot Prompt

```python
# 提供示例帮助模型理解任务
few_shot_prompt = """把中文翻译成英文:
苹果 -> apple
香蕉 -> banana
橙子 -> ?"""

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": few_shot_prompt}]
)
```

### 结构化输出

```python
from pydantic import BaseModel
from openai import OpenAI

class Recipe(BaseModel):
    name: str
    ingredients: list[str]
    steps: list[str]

client = OpenAI()
response = client.beta.chat.completions.parse(
    model="gpt-4o",
    messages=[{"role": "user", "content": "教我做蛋炒饭"}],
    response_format=Recipe
)

recipe = response.choices[0].message.parsed
print(f"菜名: {recipe.name}")
print(f"食材: {recipe.ingredients}")
```

### Prompt 最佳实践

1. **明确任务**：告诉 AI 要做什么
2. **提供上下文**：背景信息有助于理解
3. **使用分隔符**：用 ``` 或 === 分隔不同部分
4. **指定格式**：告诉 AI 输出格式
5. **分步思考**：复杂问题拆成小步骤

## 5. 参数掌握

### 核心参数

```python
response = client.chat.completions.create(
    model="gpt-4o",
    messages=messages,

    # temperature: 0-2，越高越随机
    temperature=0.7,

    # top_p: 0-1，控制采样多样性
    # 与 temperature 二选一使用
    top_p=0.9,

    # max_tokens: 最大输出 token 数
    max_tokens=2048,

    # stop: 遇到这些词停止生成
    stop=["END", "###"],

    # presence_penalty: 惩罚重复内容
    presence_penalty=0.0,

    # frequency_penalty: 惩罚高频词
    frequency_penalty=0.0,
)
```

### 参数选择指南

| 参数 | 低值 (0-0.3) | 中值 (0.4-0.7) | 高值 (0.8-1.0) |
|------|-------------|----------------|----------------|
| temperature | 确定性问题 | 日常对话 | 创意写作 |
| top_p | 保守输出 | 平衡 | 创意输出 |

### 常见场景参数

```python
# 代码生成 - 精确
{
    "temperature": 0.2,
    "top_p": 0.95,
}

# 创意写作 - 灵活
{
    "temperature": 0.9,
    "top_p": 0.95,
}

# 摘要总结 - 平衡
{
    "temperature": 0.5,
    "max_tokens": 500,
}
```

## 6. 实战项目

### 项目1：AI 对话助手

```python
from openai import OpenAI

class ChatAssistant:
    def __init__(self, api_key: str, system_prompt: str = "你是一个有帮助的助手"):
        self.client = OpenAI(api_key=api_key)
        self.system_prompt = system_prompt
        self.history = [{"role": "system", "content": system_prompt}]

    def chat(self, message: str) -> str:
        self.history.append({"role": "user", "content": message})

        response = self.client.chat.completions.create(
            model="gpt-4o",
            messages=self.history,
            temperature=0.7
        )

        reply = response.choices[0].message.content
        self.history.append({"role": "assistant", "content": reply})
        return reply

    def clear_history(self):
        self.history = [{"role": "system", "content": self.system_prompt}]

# 使用
assistant = ChatAssistant(
    api_key="your-api-key",
    system_prompt="你是一个Python专家，擅长解释代码"
)

print(assistant.chat("什么是装饰器？"))
```

### 项目2：多模型对比工具

```python
from openai import OpenAI

MODELS = {
    "gpt-4o": {"base_url": None, "key": "your-openai-key"},
    "deepseek": {"base_url": "https://api.deepseek.com", "key": "your-deepseek-key"},
    "qwen": {"base_url": "https://dashscope.aliyuncs.com/compatible-mode/v1", "key": "your-qwen-key"},
}

def call_model(provider: str, prompt: str) -> str:
    config = MODELS[provider]
    client = OpenAI(api_key=config["key"], base_url=config["base_url"])

    response = client.chat.completions.create(
        model="default",
        messages=[{"role": "user", "content": prompt}]
    )
    return response.choices[0].message.content

# 对比不同模型
prompt = "用一句话介绍Python"
for provider in MODELS:
    result = call_model(provider, prompt)
    print(f"{provider}: {result}")
```
